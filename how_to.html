Getting started

  1. check for MOF BOT on the right side bar of our discord server. image below

  2. send a DM(private message) !start to the Bot

You will be giving a new private key and a public key.You are responsible to keep your private key safe, we do not store your private keys

commands details

  1. !help

    2. this command displays all the commands available and a breif description about the commands

  2. !wallet

    3. this command displays your address that is been tracked. to set an address to be tracked use !set_wallet , note this is different from what you get from !start and !restore

  3. !start

    4. use this command to get started,a new account details(priavte key and wallet address) will be generated for your personal this, this will also enable you to use the !buyand !sell commands

  4. !orderbook

    5. this command is used to check and display all(buy and sell orders) market orders of a token by using the token market ID

  5. !set_wallet

    6. this command is used to set address to be tracked; tracked address can get notifications on new transactions,check tokens held and the worth in $(USD)

  6. !restore

    7. this command is used to restore a previous account by private key. note this will over-ride account generated when you ran !start on BOT first launch

  7. !networth

    8. this command is used to display all the token held by your personal address which was set via !set_wallet ,the token current values in $(USD) will be added soon

  8. !check

    9. this command is used to check a token info by using the token mint address

  9. !transaction

    10. this command is used to display all your last 10 transactions of your personal wallet which was set via !set_wallet

  10. !profile

    11. this command displays a mini profile of all social media accounts where your signed in, balance,private key,addresses,etc...

  11. !check_market

    12. this command is used to check a token dex price by using the token market id

  12. !buy (coming soon....)

    13. this command will be used to place a buy order on DEX

  13. !sell (coming soon....)

    14. this command will be used to place a sell order on DEX

  14. the bot listens for new transactions and notifies you (coming soon....)
